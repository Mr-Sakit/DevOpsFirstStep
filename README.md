# 🚀 DevOps Bootcamp

A documented collection of hands-on labs, projects, and practical assignments completed during a DevOps & Cloud Engineering bootcamp program. This repository reflects the skills and knowledge gained throughout the course.

---

## 📋 Learning Objectives

By the end of this bootcamp, you will be able to:

- Apply core DevOps principles including collaboration, automation, continuous delivery, and Infrastructure as Code
- Provision and manage cloud infrastructure on Azure
- Build, containerize, and ship applications using Docker
- Automate infrastructure and configuration using Terraform and Ansible
- Design and implement CI/CD pipelines with GitHub Actions
- Deploy, scale, and operate containerized workloads on Kubernetes (AKS)
- Implement observability using Prometheus and Grafana
- Deliver applications through GitOps workflows

## 📅 Weekly Overview

### 🔹 Week 1 — DevOps Foundations, Agile & Git

Introduction to DevOps principles, Agile/Scrum methodologies, and the Git version control system.

| Day | Topic | Lab |
|-----|-------|-----|
| Day 3 | PISE — DevOps & Agile Principles | Agile/Scrum tasks (PISE tasks) |
| Day 4 | Scrum Framework | Sprint planning, backlog, user stories, retrospective |
| Day 4 | Azure DevOps Boards | Lab2 — Board and task management |
| Day 5 | Git Installation | Git setup and configuration |
| Day 5 | Git Basics | Repository creation, commit, push operations |

**Key skills:** DevOps culture, Agile mindset, Scrum ceremonies, Git workflows

---

### 🔹 Week 2 — Linux, Shell Scripting & Web Servers

Linux operating system, command line, shell scripting, and web server configuration.

| Day | Topic | Lab |
|-----|-------|-----|
| Day 1 | Git Branches | Lab1 — Creating, merging, and deleting branches |
| Day 1 | GitHub Collaboration | Lab2 — Collaborating with remote repositories |
| Day 1 | Git Undo | Lab3 — Undoing changes |
| Day 2 | Linux VM | Lab1 — Provisioning a Linux VM on Azure |
| Day 2 | Linux Navigation | Lab2 — Basic navigation commands |
| Day 3 | File Permissions | Lab1 — Linux file permissions |
| Day 3 | Vi/Vim Editor | Lab2 — Editing files with Vim |
| Day 3 | System Administration | Lab3 — System administration fundamentals |
| Day 4 | Apache Web Server | Lab1 — Hosting an HTML page with Apache |
| Day 4 | Nginx Web Server | Lab2 — Nginx configuration |
| Day 5 | Shell Scripting | Lab1 — File management with shell scripts |
| Day 5 | Process Monitoring | Lab2 — Checking if a process is running |
| Day 5 | Ops Toolkit | Lab3 — Shell Scripting Ops Toolkit |

**Key skills:** Git branching strategies, Linux CLI, file permissions, Vim, Apache/Nginx, Bash scripting

| Day | Topic | Lab |
